
# Smart Workplace Mask Compliance Monitor

## Overview

This project detects whether a person is wearing a face mask using Deep Learning and Computer Vision techniques. It uses YOLOv8 for face detection and EfficientNetB0 for mask classification.

## Technologies Used

* Python
* TensorFlow
* OpenCV
* YOLOv8
* NumPy
* Pandas
* Matplotlib
* Scikit-learn

## Features

* Face detection
* Mask/No-Mask classification
* Real-time webcam monitoring
* Model evaluation using confusion matrix and accuracy metrics

## How to Run

1. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

2. Open and run:

   ```bash
   final project.ipynb
   ```
